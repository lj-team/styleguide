What it does

This tool finds styles descriptions in CSS files & outputs them to styleguide page.

Styles description looks like this:

/*
Flatbutton

Standard flatbutton

Markup:
<a href="#" class="b-flatbutton {{modifier_class}}">Link</a>
<button class="b-flatbutton {{modifier_class}}">Button</button>
<button class="b-flatbutton {{modifier_class}}" disabled>Disabled</button>

:hover - hovered
:focus - focused
:active - active
:visited - visited
.b-flatbutton-simple - tiny size
.b-flatbutton-shine - shine on hover

Style guide: Controls.flatbutton
*/

.b-flatbutton {
  position: relative;
  display: inline-block;
  margin: 0;
  ...
  }

Customize

  • config.json — contains common config. If you need live examples from some files, add their urls to section css. Files from this section will be included to styleguide page

  • styleguide.css — styles of the styleguide
